About This Project

This project challenged me to make an interactive web map. While focusing on the public transit system in Lexington, I was able to map the proximity of some of Lexington's most important public amenities: schools, hospitals, and parks. Looking at this map, it is apparant that city planners did consider transportation to many of these locations. Can you find which Lexington school has the highest enrollment rate?
